Adding - Viewing - Editing

File: web-4.png 

Adding Flow:

User clicks on Add Recipe button

Frontend sends POST /recipes with recipe data

Backend validates data

Backend saves the recipe in the Database

Database confirms save

Backend sends success response

Frontend updates the page with the new recipe


Viewing Flow:

User selects a recipe

Frontend sends GET /recipes/:id

Backend fetches the recipe from the Database

Database returns the data

Backend sends recipe details to frontend

Frontend displays the recipe to the user


Editing Flow:

User clicks on Edit button

Frontend shows editable form

User updates the recipe

Frontend sends PUT /recipes/:id

Backend updates the recipe in the Database

Database confirms update

Backend sends success response

Frontend shows the updated recipe
